## How to build and run from source?

Read the basics about extension authoring from [Extending HiTechAi Studio Code](https://code.visualstudio.com/docs/extensions/overview)

- Read [Build and Run HiTechAi Studio Code from Source](https://github.com/microsoft/vscode/wiki/How-to-Contribute#build-and-run-from-source) to get a local dev set up running for HiTechAi Studio Code
- Open the `extensions/emmet` folder in the vscode repo in HiTechAi Studio Code
- Press F5 to start debugging

## Running tests

Tests for Emmet extension are run as integration tests as part of HiTechAi Studio Code.

- Read [Build and Run HiTechAi Studio Code from Source](https://github.com/microsoft/vscode/wiki/How-to-Contribute#build-and-run-from-source) to get a local dev set up running for HiTechAi Studio Code
- Run `./scripts/test-integration.sh` to run all the integrations tests that include the Emmet tests.
